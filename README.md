An electrocardiogram (ECG) measures electrical activity of the heart. Abnormal patterns indicate arrhythmias and other cardiac conditions. Automatic classification using ML helps in early detection and decision support without sole dependence on expert interpretation. 
Wikipedia

This project:

Loads heart-beat ECG time series data

Cleans and preprocesses signals

Trains a machine learning / deep learning model

Evaluates performance with standard metrics (accuracy, precision, recall)

Saves the trained model for reuse
