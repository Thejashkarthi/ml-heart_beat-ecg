##  Project Overview

An ECG records the electrical activity of the heart. Abnormal heartbeat patterns may indicate arrhythmias or other cardiac conditions. Automating ECG analysis using machine learning helps in **early detection and decision support**, reducing dependency on manual interpretation.

This project:
- Processes ECG heartbeat time-series data
- Extracts meaningful features
- Trains ML models for classification
- Evaluates model performance
- Saves trained models for inference

---

##  Key Features

- Binary classification (Normal vs Abnormal heartbeat)
- ECG signal preprocessing and normalization
- Feature extraction from time-series data
- Model training and testing
- Performance evaluation using standard metrics
- Reusable trained model
